1. Mainwindow
    Главное окно приложения. Дочерный класс QMainWindow. Содержит в себе объекты координатной плоскости, контейнера отображения
    этой координатной плоскости и объект генератора графиков. Так же данный объект включает в себя методы для обработки нажаний кнопок и
    слотов для взаимодействия с классом создания графиков.

2. PlaneContainer
    Контейнер отображения координатной плоскости. Дочерний класс QScrollArea. Нужен для навигации по координатной плоскости.

3. CoordinatePlane
    Виджет координатной плоскости. Дочерний класс QWidget. На этом объекте происходит отрисовка координатной сетки,
    самого графика и вспомогательных элементов. Поддерживает зуммирование, отображение вспомогательных элементов (координаты), адаптированный масштаб.

4. GraphicCreator
    Объект, генерирующий графики и записывающий их в JSON файлы. Наследник QDialog. Представляет из себя форму, предлагающую
    выбрать график, его параметры и директорию сохранения JSON файла (при необходимости).

5. AbstractFunction
    Интерфейс для графиков функций. Имеет методы возвращаемого значения и имени функции. Здесь же описаны исключения, которые
    могут вызывать некоторые функции (например, достижение бесконечности). Функции создаются путем реализации этого класса.
